# iOS 26 VoIP Push Handling

## Problem/Feature Description

A team is updating an iOS 26 VoIP app and wants the current PushKit + CallKit incoming-call path. Cover iOS 26.4 behavior and earlier iOS 26.x behavior, including when a VoIP push can be ignored, when the app must report to CallKit, and the APNs server settings that matter for an incoming call invite.

## Output Specification

Create `voip-push-metadata.md` with a concise implementation outline and Swift snippets where they clarify the flow.
