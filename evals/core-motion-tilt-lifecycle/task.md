# Core Motion Tilt Control

## Problem/Feature Description

An iOS game team wants a concise SwiftUI service that uses device tilt to move
an on-screen object. They need motion updates to feel responsive without wasting
battery, and the service must clean up correctly as the view appears and
disappears.

## Output Specification

Create `core-motion-tilt-control.md` with the recommended setup, a compact Swift
example, and the implementation pitfalls the team should avoid.
