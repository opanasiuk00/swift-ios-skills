# Native Core NFC Tag Scanner

## Problem/Feature Description

Build guidance for a Swift Core NFC scanner that reads native ISO 7816, ISO 15693, MIFARE, and FeliCa tags. The team also asked whether they can use the same scanner for payment card AIDs.

## Output Specification

Create a file named `native-tag-scanner.md` containing:

- A concise `NFCTagReaderSession` implementation sketch.
- The polling options needed for each tag family.
- Session lifecycle and multi-tag handling notes.
- A clear boundary for payment-related AIDs.

Keep the answer as implementation guidance and snippets only.
