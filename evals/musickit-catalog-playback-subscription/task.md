# MusicKit Catalog Playback Flow

## Problem/Feature Description

Build focused SwiftUI snippets for an iOS 26 Apple Music search-and-play flow.
The flow should search Apple Music songs, play a selected catalog song with the
app's own player, and offer Apple Music signup when playback is unavailable.

Keep the answer scoped to MusicKit and SwiftUI. Do not build a full app.

## Output Specification

Create a file named `musickit-catalog-playback.md` containing:

- A catalog song search snippet.
- A catalog playback snippet using the app-scoped MusicKit player.
- Subscription-state handling for playback availability.
- A subscription-offer snippet with load-error handling.
