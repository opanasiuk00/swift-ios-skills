# CryptoTokenKit Security Boundary Routing

## Problem/Feature Description

An iOS team wants hardware-token certificate auth, passkeys, Secure Enclave key generation, and certificate pinning in one security milestone. They need to know which work belongs in CryptoTokenKit and which work should move to sibling skills.

## Output Specification

Create `cryptotokenkit-boundary-routing.md` with a concise ownership split and the handoff targets for work outside CryptoTokenKit.
