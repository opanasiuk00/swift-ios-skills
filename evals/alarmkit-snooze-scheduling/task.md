# AlarmKit Snooze Scheduling Guidance

## Problem/Feature Description

The alarm feature team is adding a wake-up flow to an iOS 26 app. They need a short implementation note that an iOS engineer can use to schedule a one-time 7:00 AM AlarmKit alarm with a Snooze button. The note should cover the app-side scheduling code, the setup keys and authorization flow, and the boundary between system-managed alerting UI and any optional countdown customization.

## Output Specification

Create `alarmkit-snooze-scheduling.md` with Swift-oriented implementation guidance, code snippets where useful, and a short review checklist. Do not create an Xcode project.
