A product team plans to use `mutable-content` on a silent push to decrypt the body, store the decryption key in an App Group `UserDefaults` suite, attach a downloaded media file from any URL, and use `INSendMessageIntent` for sender avatars without adding any extra capability or `Info.plist` setup.

## Output Specification

Create `rich-notification-extension-review.md` with a concise correction-focused review of service-extension triggering, secret sharing, attachment handling, communication notification setup, and service-extension fallback behavior when work fails or time expires.
