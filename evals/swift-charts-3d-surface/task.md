# Swift Charts 3D Surface Setup

## Problem/Feature Description

An iOS 26 scientific dashboard needs to render the bivariate function
`y = sin(2x) * cos(2z)` as an interactive 3D surface. The chart should have
stable axes for comparison across parameter changes, and the user should be
able to inspect the surface pose.

## Output Specification

Create `swift-charts-3d-surface.md` with:

- The Swift Charts API choice for this iOS 26 surface visualization.
- A concise SwiftUI code sketch showing the chart container and plot.
- Axis/domain setup for x, y, and z.
- How to make the 3D view inspectable.
- A note about when not to use 3D charts.
