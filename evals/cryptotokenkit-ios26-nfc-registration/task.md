# CryptoTokenKit iOS NFC Smart Card Plan

## Problem/Feature Description

An iOS 26 app needs to read a contactless PIV smart card over NFC and keep a token-backed keychain reference for later signing.

## Output Specification

Create `cryptotokenkit-ios-nfc-plan.md` with the relevant CryptoTokenKit APIs, availability checks, session lifecycle, registration approach, and keychain lookup behavior.
